This repository contains a collection of parametric SolidWorks models developed from beginner to intermediate level, focusing on building strong fundamentals in CAD modeling and design intent.

About this project 
These models are created to understand how parametric design works in SolidWorks, where geometry is controlled using dimensions, equations, and constraints. The project helps in developing the ability to modify designs efficiently without rebuilding from scratch.
